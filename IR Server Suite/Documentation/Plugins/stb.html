This list is un-tested, as I do not own any of these set top boxes I cannot confirm they work. Most of these set top box codes have been provided by users, most have captured codes with a Microsoft MCE IR receiver, which means most of these codes will not work with other IR hardware. Some will work because they are in Pronto CCF format, which is compatible with many IR Blasters.

Some of these set top boxes have generic names, like "Sky", because they will work for multiple models of set top boxes or because the exact model number is unknown. If you find a problem with a setup, or find some settings missing, please email or forum post the details to me. If you have a set top box that is not on the list and you have it working, please email or forum post the IR files and details to me.
