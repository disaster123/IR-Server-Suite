Learn IR Command


When you create a new IR or re-learn an IR command this window will appear.
  • If you're teaching a new IR command, then you will need to specify a name. This name will be used to create a file, so make sure the name supplied doesn't break any of the windows file name conventions. That is, don't use / ? < > \ : * | "
  • Select an output port (if applicable) and click "Test" to ensure the IR Command was learned correctly.
  • Click "Learn" to teach the plugin your IR command. After clicking "Learn" you point your remote at the IR device and press the button to learn. Follow all the usual procedures for teaching IR commands. That is, avoid direct sunlight on the receiver, stay away from plasma tv's, and turn off any fluerescent lights. All these things can interfere with the InfraRed signals. Place your remote within a couple of centimeters (within an inch) of the receiver to ensure good reception of the signal.
