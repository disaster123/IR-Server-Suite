Macro Editor


This window is for setting up and modifying Macros.

Name:
Just as the case is with IR commands, Macros must have a name that conforms to windows file name conventions.

Macro:
In this group of controls you lay out the order of commands in the macro. The macro is executed from top to bottom. Click Up / Down to move a command within the list. Click Remove to take a command out of the list.

Commands:
This is a list of available commands. Click "Add" to add the selected command to the macro.

You can click Test at any time to test your macro so far. When you are done click OK.

Available Commands:
Subject to availability with the particular plugin or application you are creating the macro, the following commands will be available for inclusion in the macro ...

  • IR Commands
  • External Program
  • Serial Command
  • Message Command
  • Keystroke Command
