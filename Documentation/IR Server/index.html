IR Server


This is the IR Server configuration window which is accessed by the right-click context menu of the IR Server tray icon.

Device plugin:
Choose the IR Device you want to use for receiving and the device you want to use for transmitting (blasting) by placing a mark in the box under the appropriate column. By double-clicking the device name in the device plugin list you can select both receiving and transmitting cabilities for that device at the same time. Note that you do not need to have a device selected in either column, that is, you can run the IR Server without a transmit, receive or either type of device enabled.

Click "Configure" to set any device specific options.

Mode:
This option sets the mode that the IR Server will operate in.

  • Server mode - This is the default option where the program operates as a central server hosting the locally connected IR device's capabilities to other plugins and applications in the suite.
  • Button relay mode - When in this mode the server will relay button presses recieved from the IR device to another IR Server.
  • IR repeater mode - This mode has the server running as a repeater for another IR Server. When the host IR Server transmits an IR command the repeater mode server will also transmit the same IR command.

Start IR Server with Windows:
Select this option to have IR Server start automatically when windows boots up.
